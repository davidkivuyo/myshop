# Myshop
One can store his/her shop sales, new products and debts during business. good for small businesses which needs storing of many of its operational details.
# Getting started with myshop
* Download XAMPP from apache friends
[Download XAMPP for any operating system](https://www.apachefriends.org/download.html)  and then install it.
* Start apache server and mysql  <img width="1276" height="673" alt="Screenshot 2026-02-07 111227" src="https://github.com/user-attachments/assets/b7598d02-f7f3-4758-8faf-bdbd006d9cb9" />
* navigate to your directory where you installed xampp , find and open 'htdocs' folder
* inside 'htdocs' folder create a folder named 'myshop' <img width="1069" height="217" alt="Screenshot 2026-02-07 104325" src="https://github.com/user-attachments/assets/52b4e60f-1e74-427c-85ba-8afa7c42b523" />

* open the terminal inside the 'myshop' folder <img width="1114" height="655" alt="Screenshot 2026-02-07 120609" src="https://github.com/user-attachments/assets/045d646f-fb99-47bc-ae9f-a374bf333fb5" />

* inside the terminal run the following command   ```git clone https://github.com/davidkivuyo/myshop.git```
